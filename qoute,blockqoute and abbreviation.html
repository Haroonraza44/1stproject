Blockquote < blockquote>

the content of the <em > element is shown in italics, and a blockquote is usually indented. But you should not use them to change the way thatyour text looks; their purpose is to describe the content of your web pages more accurately.The reason for using these elements is that other programs, such as screen readers or search engines, can use this extra information. For example, the voice of a screen reader may add emphasis to the words inside the em element, or a search engine might register that your page features a quote if you uses the < blockquote > element.
